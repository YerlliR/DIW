HTML5

Veremos los nuevos elementos estructurales, así como los nuevos elementos existentes para formulairos, audio y vídeo. El nuevo elemento canvas nos abre un mundo de posibilidades en el desarollo de videojuegos web.

CSS3

Mejoraremos nuestros diseños web utilizando las nuevas características de CSS3: efectos, transformaciones, animaciones, etc. Con CSS3 nuestros diseños serán más simples, pero no menos impactantes.

JavaScript

Junto con HTML5 aparecen nuevas API javascript que nos aportan multitud de facilidades y posibilidades en el desarollo de nuestros proyectos web. Con las nuevas API javascript nuestras manos dejarán de estar atadas.
